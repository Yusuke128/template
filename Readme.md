# template

このテンプレートは、LP / コーポレートサイト / WordPress テーマ開発を
高速化するためのベーステンプレートです。

## 主な内容

- SCSS によるモジュール化構成
- fluid font（clamp）対応のタイトルコンポーネント
- カードレイアウト / グリッド / ボタンなどの汎用 UI
- WordPress テーマのベーシック構成  
  （functions.php / index.php / style.css / screenshot.png）

## 使い方

このリポジトリをコピーして、コーディング・WordPress テーマ作成を進めます。
